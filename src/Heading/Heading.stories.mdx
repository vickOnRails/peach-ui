## Quick Usage
The fastest way to use the `Heading` component is to add the required `level` props. 
The `Heading` component can take any of the levels `h1` | `h2` | `h3` | `h4` | `h5` | `h6`. 

### level
Set the `level` of the `Heading` by passing the appropriate heading level

<Preview>

    <Divider/>

    <Divider/>
    <Heading level="h5">Heading 5</Heading>
    <Divider/>
    <Heading level="h6">Heading 6</Heading>
  </>
</Story>

### size
You can also control the size of the `Heading` by customizing the `size` prop irrespective of the heading `level`. 
This is useful in cases where you want a `h6` to appear large.

Setting the `centered` prop to `true` centers the heading.
