## Quick Usage

The easiest way to get text on the screen is to use the `<Text/>` component

## size

Use the `size` prop to customize the size of the `Text`. Values include

<Preview>

      <Divider/>
      <Text size="display2">Display 2</Text>

      <Text size="display3">Display 3</Text>
      <Divider/>
      <Text size="heading1">Heading 1</Text>
      <Divider/>
      <Text size="heading2">Heading 2</Text>
      <Divider/>
      <Text size="heading3">Heading 3</Text>
      <Divider/>
      <Text size="heading4">Heading 4</Text>
      <Divider/>
      <Text size="heading5">Heading 5</Text>

        Lorem ipsum dolor sit amet, consectetur adipisicing elit. Aperiam

        iste, harum dicta maxime. Labore aliquid commodi veniam nulla sit nemo
        officiis.
      </Text>
      <Divider/>

      <Text size="small">Small</Text>

Pull a `Text` paragraph to the center by setting the `centered` prop to `true`
